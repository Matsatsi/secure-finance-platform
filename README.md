# secure-finance-platform
Enterprise microservices platform built with Java 21, Spring Boot, React, PostgreSQL, and Docker.

Enterprise microservices system built with:

- Java 21
- Spring Boot
- React
- PostgreSQL
- Docker
- Microservices Architecture

## Services

- User Service
- Account Service
- Transaction Service
- Fraud Detection Service
- Notification Service

## Architecture

API Gateway → Microservices → PostgreSQL
